## 合成表

### OneDrive

进阶路线：

1. Onedrive + Openlist = 云盘聚合
   官方文档很详细了：[快速开始 - OpenList 文档](https://doc.oplist.org/guide)

   【提示：咸鱼上5T永久空间不靠谱，博主存的文件全没了】![image-20260429164431427](https://cdn.218501.xyz/2026/04/9118231a7fbc464ee1f26f8a4ee90e81.png)

2. 云盘聚合 + Rclone + Navidrome  = 音乐网站
   ![image-20260429164806222](https://cdn.218501.xyz/2026/04/d2505ea112108f847a346f2418706bb7.png)

3. 音乐网站 + Musiver = 音乐软件

### CloudFlare

1. CloudFlare + cloudflare_temp_email = `无限邮箱`

   看LinuxDo大佬的文章，特别详细：[【教程】2026版 小白也能看懂的自建Cloudflare临时邮箱教程（域名邮箱） - 文档共建 - LINUX DO](https://linux.do/t/topic/1666961)
   ![image-20260429164915646](https://cdn.218501.xyz/2026/04/6d4645d80f9700a6f9431561c91eb7dc.png)
2. CloudFlare (Page + Snippet) + Static_RandomPicAPI = `随机图`

   示例网站：[Random Pic API](https://pic.ssr.ddns-ip.net/)

   搭建可参考二叉树的Github项目：[afoim/choice_randompic: 和 https://github.com/afoim/Static_RandomPicAPI 搭配使用，通过Snippet使得服务端返回302的随机图](https://github.com/afoim/choice_randompic)
   ![image-20260429165116099](https://cdn.218501.xyz/2026/04/c58c86d1618f9a7d6e7e045a631fa1e1.png)
3. `无限邮箱` + CLIProxyAPI + CC Switch = `Codexhao号池`
